木秀于林风必摧之与反脆弱

三国时文学家李康在《运命论》中提及

“故木秀于林，风必摧之；堆出于岸，流必湍之；行高于人，众必非之。前鉴不远，覆车继轨。”

说的是在大风来临时，最易摧折的必是那片树林中最突出的树；故应了儒家的恭敬谦让，藏锋守拙不出头，视为保护自身的重要方式
凡事利弊相依，往往这时选择隐身人后，便丧失了台前实践试错的机会，无法在挫折磨砺中迎来新生
而反脆弱性，则与之相反，是在不确定性和变化中寻找机会，接受并利用不确定性，而不是避免它，正如塔勒布在书中提及的九头蛇怪

九头蛇怪是一种住在阿尔戈斯附近勒拿湖中的生物，长着数不清的头。每次有一个头被砍，就会重新长出两个头来，其危害可想而知。九头蛇怪就是反脆弱性的代表

始终坚守成长性思维，在当下的圈子中积极大胆去尝试做一些具有挑战性的事项，以及勇于去直面困难；毕竟人是活的，木秀于林会遭受摧折，而人秀于林则会到达一片更高更好且更具有包容性的下一片树林，不要在意当下的挫折带来的不利影响，要站在时间跨度更长的视角去回顾这一切该不该做

我的经验与教训

Posted on 2024-05-15 Edited on 2025-01-01 Word count in article: 1k Reading time ≈ 3 mins.

在许单单的《临界点》一书中读到黑石苏世民的25条工作生活原则，兴趣而来便翻读了《苏世民：我的经验与教训》，还没读完，结合当下已阅读的部分联系自身实际，分析下当下的困境和原因，以及解决方案

  • 从“这对我有什么用”到“我可以为他人做什么”的思维转变
  • 如何继续沉溺于当下的职场困境
  • 轨内草和轨侧草

从“这对我有什么用”到“我可以为他人做什么”的思维转变

  • 人们最感兴趣的话题永远是“自己的问题”，苏世民书中提到

    许多人失败的原因是从自身利益的立场出发，只选择性的听取与自己有关的话题，至于其他的话题总觉得“这对我有什么用”。取而代之的方式，去仔细聆听对方谈话的内容，认真观察别人表达的方式，能帮助找到“我能提供什么帮助”这一问题的答案，从而可以帮助别人，成为解决问题的朋友，其他的一切都会随之而来。

  • 回到工作，当下直属leader的困境何在？我能提供哪些帮助。对整体的风控体系、三方组件功能、稳定性保障等方面不够熟悉和风险无法识别，是否可以整体相关材料并用通俗的语言深入浅出的表达出来；理清楚当下的进展和已经做的事情，列出需要寻求帮助的部分；针对一些工作内容及提效，团队发展抛出问题和给出差异的决策方案，降低其决策难度
  • 对于交友和情感方面而言，以个人利益为中心导向的发展只会加深索取从而失衡，从可以为他人做什么的思维出发，提供一个新的视角，解决他人的困境方能收获友谊情感
  • 对于社会而言，能给他人提供什么什么样的价值，自然便是商机

如何继续沉溺于当下的职场困境

  • 继续被“需要激励”蒙蔽自我，从而持续沉溺于消极应付工作
  • 继续“装”躲身于后台，不去积极利用公司资源和团队杠杆做更有影响力的事情，干自己那没有一丁点壁垒的一亩三方田之事
  • 继续投身于琐事杂事，日复一日享受清理掉一件又一件小事的愉悦感，而不敢聚焦于真正的关键重要事项，避难就易
  • 继续重复踩坑，不做任何事项的任何复盘，每次任务都是新的开始毫无积累，持续保持对大项目任务的恐惧感
  • 继续不分主次，聚焦于做一堆低价值杂活，以满足自己每日的充实感
  • 继续不做每日任务拆解和时间记录，不去形成日常任务有效的数据落存，从而随意的分配时间，随意接受和拒绝任务且无法提供有效数据支撑

轨内草和轨侧草

每日通勤乘坐轻轨，透过车窗观察发现轨道内和轨道两侧都有杂草丛生，居于轨道内的草日日车来车往饱受摧残无法成长壮实，而轨道两侧则欣欣向荣；正如史记中秦李斯的厕鼠和仓鼠论如出一辙；环境因素亦十分重要，居于轨道内之杂草和厕鼠，即使拼命努力也无法有好的结果，提醒自己在一个尝试过努力并且仍不能有所作为的环境中时，换个环境也是一个解决思路。正如苏世民早期在帝杰证券不尽如人意的经历，及时脱身止损未免不是个好选择。

背景

回顾近期及以往不预则废的日常生活，期望于年底前推进一些事项的落地，于是便开启这每日一篇博文小项目，并借此输出强化思考和促进行动的改变
推进事项

  • 认知提升，高效决策和行动
  • 拓深事业护城河，稳定收入来源支持开销和应急（编程和外语等）
  • 人生丰富体验（有限的一生体验丰富的生活，了解世界的丰富多彩）

如何完成每日一篇认知博文

  • 固定的输入时间，充分输入是有质量的输出前提（如每日上下班通勤时间+午休部分时间等）
  • 借鉴“熔断不读书法”，发掘一个与以往认知冲突或触动自己的点，拓展由点及面，完成博文内容的充实和后续实践
  • 笔记整理，使用幕布思维导图式APP，随时记下灵感与归纳，思考与总结；作为当日博文的重要素材
  • 坚持写下来，无需上来就关注对他人是否有价值，最关键的是帮助自身完成知识积累，认知迭代，先助己再助人，对自己能提供有效帮助的，自然也能为像自己一样仍在认知学习路上的人有帮助
  • 奖励，每个里程碑的达成给予自身一个切实的激励；
  • 时刻明确自己目标，为什么要投入时间做这件事，它的机会成本是什么样？有其他更值得去做的事情吗

认知小课堂

自我剖析，缓释焦虑与消除模糊

本能脑、情绪脑、理智脑的三重大脑

  • 由于本能/情绪脑的天性，人们往往做出短视的决策和行动，追求趋易避难和急于求成
  • 在有不符合理智的想法和行为时，陷入自我解释，企图对自身行为欲望合理化

缓释焦虑

  • 探焦虑之因
    • 想同事做多件事，又急于看到结果
  • 破焦虑之贼
    • 正视问题
      • 列出产生焦虑的事项，分析其本质，依次行动去解决，而不是避而远之，逃避问题
    • 节奏可控
      • 拉伸区边缘着力，过易低水平勤奋提升有限，过难遭遇挫折止步不前
    • 成事
      • 要事优先
      • 做减法，一段时间内只聚焦于一件事
      • 价值产出导向

消除模糊

消除模糊，亦即追求确定，能够掌控的状态

  • 寻模糊之源
    • 目标模糊 一日不清晰便挥霍一日
    • 选择模糊 无视机会成本下的肆意豪赌
    • 行动模糊 弃马徒步下的横冲直撞
    • 反馈模糊 是不是做正确的事只有天知道
  • 破模糊之道
    • 正视，拆解，实践复盘，迭代
    • 思堵力行，力竭谋局；无需刻意等想好了再做，做完了再分析；边学边做，边做边想
    • 用正确姿势投丢的球，比用错误姿势投丢的球更有价值；亦即一件做对的事要学会快速复制，提升影响力和边际收益

读成甲《好好学习》，带着几个问题思考，亦即思考他人经验于已有何复用之处，或者是有何启发

如何陷入“低水平勤奋陷进”

  • 忽视价值，盲目追求量，每年读个100本书，写10w行easy代码，只求舒适圈量的输入而不去思考有何价值产出
  • 忽视关联，不去深究本质，不去链接已有知识
  • 忽视输出，只摘抄不复述，不用自己话表达出来
  • 忽视反馈，不去在实践中进行验证，不去寻求有效反馈

最近做过哪些习惯性防卫

  • 上周研发同事问测试遇到问题是否先排查问题？当即回复是不是在质疑我能力
  • 上周同组同事录制压测流量咨询问题时，在被给出其思路未解决问题过来再次确认时，马上紧张立刻开始排查，要确认问题就是其操作流程有问题。虽然最终确认开始给出的排查思路是对的，但是未去反思，对于一个新手而言，自己之前输出的操作文档而言是不合格的，应该去改进操作文档使其更有效，而不是排查出问题立马群里反击
  • 周日科三第一次学习，教练反复提示挂挡，油门踩刹等，因为是偏指导性，自己没什么反驳空间，所以只是偏解释型的习惯性防卫。从事后去反思角度来说，可以去提前网上视频自学下，第一次课的效果会更好些

故习惯性防卫，虽然是出于本能短时间内改善难度大，但是也是可以改的，如何才能改，首要的就是记录下每日习惯性防卫的场景有哪些，其次花一定时间静心去分析其背后做出此类行为的本质是，即思维方式是什么样的，这种思维方式是正确的吗，如果是存在谬误的，那再从思维到行为到习惯进行改进

不做日常反思会怎么样

  • 每日依旧充斥着不合理的假设（预期），与实际差异过大时则沮丧焦虑，或者是跌落至舒适圈享受低水平勤奋带来量的直观感受
  • 设定的长期目标（如6month的项目目标），缺乏进度条而过早放弃，或对拆分的阶段性里程碑和todo事项缺乏预期实际对照，缺乏有效反馈，从而无法及时调整导致失败
  • 弊病依旧，在职场生活中被认为是思维固化的人，沟通表达不清晰，难以达到自己预期的目的，抱怨工作和因矛盾去埋怨家庭亲友
  • 无法聚焦于真正有价值产出的部分，疲于应付琐事急事，满足于低效率的充实，无法形成个人有效竞争力，或者缺乏去做有意义事情的时间
  • 遇到重大挫折困难手足无措、悔恨不已，导致这种情形其中来自自己的原因的部分，是可以在日常事项中辨识，如果去做好每日反思，及时改进自身的错误假设，和补偿其带来的已有损失，或许就可以扭转导致不利结果的趋势，降低带来的风险和伤害

怎么去做以教为学

  • 范围，工作方面将工作思维方式、工作技巧、调研新知识，在组内去传递；生活方面结合自身经验思考沉淀去给出有建设性意见惠及亲友；博文输出启发他人思考
  • 寻求同频或准同频认知思维的个人团体作为分享对象，这样分享内容的实际反馈更有效，能触发深刻认识快速迭代，如测试思维技能等和组内同事分享，而不是和业务产品其他角色去分享
  • 不断调研和实践落地，学习掌握有价值的内容，然后去教授给他人，而不是分享一些低价值的东西去浪费他人时间
  • 前期注意以高频输出打磨技巧，并汲取复用他人已有优秀经验

刻意练习应该怎么做

  • 拉伸区的反复实践，有80%的精力时间在拉伸区
  • 利用反思方式去核对范围和投入，以期改进，可以有个逐步的改进过程
  • 充分利用暗时间，运用潜意识去强化学习
  • 在某个需要改进的小范围加速迭代的频次，充分利用复利效应，快速达成自己的预期
  • 目标的唯一，在项目推进的阶段时刻做减法，降低切换任务导致的专注力转移，能够专注于目标持续精进

  2. 久久萦绕的杂念
  3. 预期与实际存在较大差异的事项
  4. 新认知运用实践的预期与实际分析 、

从零开始搭建思维框架

Posted on 2022-12-03 Word count in article: Reading time ≈ NaN:aN

为什么要搭建思维框架

当下工作、生活决策依赖于已有经验，在未涉及或经验薄弱之处手足无措，并且有些经验未有过多高价值的反馈，其准确性不知；故构建自身的思维框架，去主动构建有利于整合输入的认知知识以及倒逼输入，靠思维框架去决策有利于快速迭代，保持与时俱进不固化思维；大量实践也有利于形成对未知事项有分化处理有效评估影响的能力，从而能够从容冷静面对困惑，从多角度高层次看待问题，甚至于长期角度去思考决策，做有利于长期利益之事

如何搭建思维框架

既然是搭建的操作，便有预计目标，可以用SMART原则指导
——S代表具体(Specific)
是具体的，一方面是思维框架应用场景，另一方面是底层原则，二者关联做映射，决策实践去验证其有效性

  • 要搭建什么样的思维框架-能够运用于日常工作生活中各项决策，提升决策效果（结合反思日记和现实实际反馈中实际低于预期频次和严重程度）
  • 持续多长时间，周期是什么样的-长期，随时update，月/季度周期去回顾整合
  • 达到什么样的效果-能够在遇到未知事项前，不是盲目迎难而上，而是能识别风险有清晰思路掌控住每个环节，或者在出现不利局面时能快速扭转

——M代表可度量(Measurable)
可以度量，将底层原则不断纳入思维框架应用场景，并去反复校验它，最终形成一个清晰完备的思维框架及应用场景和底层原理，哪些场景可以应用哪些底层原则
到底什么样才是有效，采用思维框架前后到底有没有差异，以前经验决策中是否蕴含底层原则，能否识别出来并纳入思维框架，在后续同性质或其他场景能否复用？
——A代表可实现(Attainable)
可以实现，识别过往经验中底层原则+习得他人优秀的经验（模仿，阅读，交谈）+输出实践反馈（给他人带来价值，从实际结果鉴别有效性）
——R代表相关性(Relevant)
有相关性，一是自我认知提升的重要尝试，二是这是一件能给他人带来价值的事，如果自身可以有效搭建思维框架提升解决困惑能力那便对他人也有借鉴意义
——T代表有时限(Time-bound)
有时限，每个迭代周期是有时限的，螺旋上升，3个月完成思维框架V1.0搭建，识别或吸收18+个底层原则，完善10+思维框架应用场景梳理，50+细分场景梳理和底层原则映射，在12个细分场景下反复用映射的底层原则去决策执行，并分析其反馈及预期实际差异和待改进部分

搭建思维框架做哪些事？

  1. 梳理思维框架应用场景（生活、工作）
  2. 每天思考5-10件事的决策执行中，运用了哪些底层原则，如果没有运用底层原则又是什么原因？
  3. 每周将习得的1-2个底层原则纳入思维框架场景中
  4. 每周关注1-2个场景，采用从经验中识别出的或习得的底层原则，关注每次决策执行过程的反馈如何？
  5. 考虑整个思维框架构建过程中哪些是有价值的部分，如何能复制给其他人？
  6. 如何能教会指导别人也采用思维框架或其中底层原则去处理生活工作场景，提升自我影响力

我是怎样被说服做一件事的？

Posted on 2022-11-07 Edited on 2022-11-08 Word count in article: Reading time ≈ NaN:aN

哎，我是怎样被说服做一件事的？

今日晨约5时醒，冲澡早饭后逐渐清醒，开启晨读；当翻开书本前就意识到，又是由“急于求成”的想法导致了周日浑浑噩噩的一天，没有实质的输入和实践，便急于成效反馈，故预期不达焦虑不安
稍读几章便顿释焦虑，好奇为什么这些文章如此有效，并且我是如何被说服的去接受作者的观点；静心分析一番，发现作者每个观点后都列举出一项项做这件事的好处，而且多从作者自己经验出发实际落地的实践；按这些思维方式及行为去探索实践，带来的好处是切实符合自己利益述求，故而自然并无抵触欣然接受

哦，那么如何才能说服一个人去做成某件事呢？

那么如何才能说服一个人去做成某件事呢？或许这其中的核心部分便是私利
信息需要传递，也需要沟通交流传递观点，使目标达成，要多人合作，达成某个目标，那不可避免就要在意见不一时求同存异，采用说服的方式，那么说服有哪些场景

  • 不说服，目标达不成；或者这部分活自己干了
  • 说服，仅用技巧或情侣间甜言蜜语达成自身目的，不涉及对方私利；当下或许接受了，醒悟时刻倍感受欺骗
  • 说服，用私利不用委婉技巧；一句“都是为你好”，往往家长教育子女虽说经验丰富建议中肯，但不容易接受且被抵触
  • 说服，私利+技巧，达成目的

战国策《触龙说赵太后》是个很好的例子，秦急攻赵，赵氏求救于齐，曰必以长安君为质。长安君为赵威后的小儿子，爱子心切，岂肯为质异国受苦；触龙的进言很巧妙，先是言自身年老，行步饮食关切争取同情；再是荐少子补黑衣之数，以卫王宫，博得爱怜少子共鸣；又是以爱燕后贤于长安君巧设悬念烘云托月，从而最后点出“位尊无功，奉厚无劳，—旦山陵崩，难以自托”的困境，彻底的说服了赵威后；前面铺垫为了点出最后的私利部分，便是今令质齐有功于国，而得自立于赵

说服常常用于同等沟通，上对下沟通往往交待便是，下对上沟通说服往往需要考虑周全。如臣子对主上进言，韩非的《说难》一文提及了诸多场景很值得细思-太深刻了，佩服（怎么“舔”出层次）

  • 追求名高说以厚利 -> 进言者被视为节操低下，并受到卑贱待遇，必然会遭人抛弃、被人疏远
  • 追求厚利说以名高 -> 进言者显得没有心计， 而且脱离实际，其建议也必然不会被采纳
  • 阴厚利显名高，说以名高 -> 会在表面上采纳进言者的意见，而在实际上疏远进言者
  • 阴厚利显名高，说以厚利 -> 会在实际上采纳进言者的主张，却在表面上疏远进言者
  • 知饰所说之所矜而灭其所耻 -> 懂得粉饰他自夸的事而掩盖自耻的事
  • 有私急也，必以公义示而强之 -> 有私人的急事，就指明是合乎公义的鼓励去做
  • 其意有下也，然而不能已，说者因为之饰其美，而少其不为也 -> 有卑下念头不能克制，那就粉饰成美好抱怨不去干
  • 心有高也，而实不能及，说者为之举其过而见其恶，而多其不行也 -> 有过高的企求实际不能达到，就举出此事的缺点揭示坏处，称赞他不去做
  • 欲矜以智能，则为之举异事之同类者，多为之地，使之资说于我，而佯不知也以资其智 -> 想自夸才智，就为其多举别的事情中同类之事，多给他提供根据，让他从我这里借用说法，而我却假装不知道帮助他自夸才智
  • 欲内相存之言，则必以美名明之，而微见其合于私利也 -> 想进献与人相安的话，就必须用好的名义阐明它，并暗示他符合其私利
  • 欲陈危害之事，则显其毁诽而微见其合于私患也 -> 想要陈述危害的事，就名言此事会遭到的毁谤，并暗示对他也有害处
  • 誉异人与同行者，规异事与同计者 -> 称赞另一个与他行为相同的人，规划另一件与他考虑相同的事
  • 有与同污者，则必以大饰其无伤也 -> 有和他污行相同的，就必须对它大加粉饰说它没有害处
  • 有与同败者，则必以明饰其无失也 -> 有和他败迹相同的，就必须对它明言掩饰，说他没有过失
  • 毋以其难概之，无以其谪怒之，毋以其败穷之 -> 不用为难的事压抑他，不用过失的事激怒他，不用败绩去困窘他
  • 二人说者皆当矣，厚者为戮，薄者见疑，则非知之难也，处知则难也 -> 武公伐胡，富人失窃，说的直白被杀，说的间接委婉被猜疑，是处理所了解的情况困难
  • 弥子之行未变于初也，而以前之所以见贤而后获罪者，爱憎之变也。故有爱于主，则智当而加亲；有憎于主，则智不当见罪而加疏 -> 宠爱时受亲近；弥子瑕色衰爱弛受憎恶时，才智就显得不恰当，遭到谴责而更被疏远

其实不单单臣子对主公沟通，其实现实生活中弱势对强势，正如感情中处于低位的一样，试图仅靠真诚相待获取信任往往事与愿违，或许唯有察其爱憎兼慎言，旷日离久周泽既渥后方能平等交流；当然也有planB，提升自己或者换个人

1、荀子的慎言

故不问而告谓之傲,问一而告二谓之囋

慎言，急躁和啰嗦都不可取。很有道理，如果是一篇博文内容多，他人大可跳过不读，如果是现实交谈，话多既浪费彼此时间，又可能让他人觉得你啰嗦显摆，或者给其带来被轻视感觉

2、梁惠王的反思

东败于齐,长子死焉;西丧地于秦七百里;南辱于楚

敢于反思正视自身的人物值得学习，反观自身，造遇困窘时常有掩饰逃避想法念头，实不可取，可以从反思日记着手实施；by the way，梁惠王弃商鞅不用也可以理解，毕竟像公司大了采用激进的方式变革转舵风险太大
